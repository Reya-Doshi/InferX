# InferX Support Guidelines

We are dedicated to providing support for the InferX open-source project. This guide outlines how to find help, report issues, and discuss features.

---

## 1. Finding Help

*   **Documentation:** Review our structured guides and API references in [docs/](docs/).
*   **FAQ:** Check the list of frequently answered questions in [FAQ.md](FAQ.md).
*   **GitHub Discussions:** Ask questions, share design feedback, and showcase your deployments under our GitHub discussions tab.

---

## 2. Reporting Issues

*   If you find a bug, please open a GitHub issue using our [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md).
*   For feature requests, use our [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md).
*   For security concerns, please refer to [SECURITY.md](SECURITY.md) and do not post publicly.

---

## 3. Support Tiers

InferX is community-driven. If you need dedicated SLAs or custom engine integration support, contact `support@inferx.ai` to inquire about developer support tiers.
